# Pizza_Sales_Dashboard
This repository contains a Power BI dashboard built to analyze pizza sales performance over a one-year period. The report uncovers key insights such as revenue trends, busiest days, category-wise performance, and best/worst-selling pizzas to support data-driven decision-making for restaurant operations.

# Features
KPI cards for Total Revenue, Average Order Value, Average Pizzas per Order, Total Orders, and Total Pizzas Sold
Daily and Monthly order trend visualizations
Sales performance by pizza category and pizza size
Best- and worst-selling pizzas by revenue, quantity, and total orders
Category-wise total pizzas sold
Interactive filters for pizza category and date range

# Overview
This project simulates the role of a Business Analyst for a pizza restaurant, helping identify peak sales periods, customer ordering patterns, best-performing menu items, and opportunities to improve store operations. It serves as a practical example of real-world BI reporting and storytelling.
